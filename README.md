# My Portfolio 

Ein vollumfängliches, responsives Portfolio-Projekt, das als digitale Visitenkarte und Showcase für meine Bewerbung um ein Pflichtpraktikum dient. Das Projekt wurde von Grund auf neu strukturiert und erweitert, um nicht nur meinen Werdegang, sondern auch konkrete Code-Beispiele und Projekte interaktiv darzustellen.

## Technologien (Tech Stack)
* **HTML5:** Strukturierung der Webseiten (Semantic HTML).
* **CSS3:** Flexbox, CSS Grid und globale CSS-Variablen für ein konsistentes, responsives Layout über alle Unterseiten hinweg.
* **Architektur:** Multi-Page-Application (MPA) Ansatz ohne externe Frameworks (First-Principles).

## Projektstruktur und Navigation
Das Portfolio ist modular aufgebaut und gliedert sich in folgende Hauptbereiche:

* `index.html` - Die Hauptseite mit Übersicht über Fähigkeiten, Werdegang und direkten Links zu den Zeugnissen.
* **Fachspezifische Showcases:** Detailseiten mit Code-Ausschnitten (Snippets) und Zusammenfassungen der erlernten Konzepte:
  * `java.html` - Java Grundlagen und OOP-Konzepte.
  * `python.html` - Python-Projekte (z.B. Password Generator, Tip Calculator).
  * `php.html` - PHP & serverseitige Logik (Formularverarbeitung, Sessions).
  * `frontend.html` - Frontend-Entwicklung (HTML, CSS, JavaScript).
* **Rechtliches:** Standardkonforme Seiten für den deutschen Raum:
  * `impressum.html` - Anbieterkennzeichnung.
  * `datenschutz.html` - Datenschutzerklärung.

## Dateisystem
* `icons/` - Vektorgrafiken für Technologien.
* `bild/` / `Fotos/` - Profilbilder und Assets.
* `PDF_Files/` - Hinterlegte Zertifikate, Bewerbungsunterlagen und Projektbeschreibungen.

## Projekt starten
1. Klonen Sie das Repository oder laden Sie die Dateien herunter.
2. Öffnen Sie die Datei `index.html` in einem beliebigen modernen Webbrowser (kein lokaler Server erforderlich).
